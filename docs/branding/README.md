# Yamtrack Branding

This folder contains the current shipped branding assets for the iOS app.

## Current Source Of Truth

The current Yamtrack mark is the square raster asset in `../../yamtrack-ios-logo.png`.

It is the approved source for:

- the iOS app icon exports in `YamtrackiOS/Assets.xcassets/AppIcon.appiconset`
- the current standalone branding preview in `png/yamtrack-mark.png`
- the compatibility lockup preview in `png/yamtrack-logo.png`

## Current Direction

The mark is the canonical iOS logo and should match the app icon outputs shipped from this repository.
